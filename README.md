# The News Room

A modern news portal built with React, Vite, Tailwind CSS, and React Router.

## Features

- Latest News
- Category-wise News
- News Details Page
- Authentication
- Responsive Design

## Technologies Used

- React 19
- Vite
- Tailwind CSS
- DaisyUI
- React Router
- Firebase
